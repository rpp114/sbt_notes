How to Build Variables:

There are a variety of pre-built variables that you can use to pre-populate the report text. To place the value in the report text, use the variable signifier. IE: to see the client's first name in the report put {first_name} in the text.


Available Pre-Built Values:

first name: {first_name}

last name: {last_name}

age: {age}

gestational_age: {adjusted_age}

boy/girl: {child}

he/she: {pronoun}

his/her: {possessive_pronoun}


In order to create an list of the assessment section responses, put // before and after the score of the responses of interest. IE: //score_0// will create a list of all the assessment responses with the score of 0. For Cargiver Responses: //caregiver_score_0// will create a list of all the assessment caregiver responses with the score of 0.
